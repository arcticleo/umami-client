# Claude Development Instructions

## Working Style

**IMPORTANT**: When implementing this project, work in small incremental steps. Complete ONE step at a time from the implementation plan, then check in with the user for confirmation before proceeding to the next step.

The user wants to follow along and learn during the development process, so treat this as both a development task and a tutorial. After each step:

1. Explain what was done and why
2. Wait for user confirmation
3. Only proceed to the next step after approval

This approach allows the user to:
- Better understand each phase of development
- Ask questions about specific implementations
- Learn Ruby gem development patterns
- Have control over the pace and direction

## Project Context

This is a Ruby client gem for the Umami Analytics API. See IMPLEMENTATION_PLAN.md for the full specification and step-by-step implementation details.
